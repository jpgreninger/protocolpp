Some fields may only be available in the jicmp protocol due to their dynamic nature as fields in the ICMP header or packet and are not stored in the security association. For ICMP they include

  • CHECKSUM
  • POINTER
  • TYPE
  • SEQNUM
  • GATEWAY
  • ORIGTIMESTAMP
  • RXTIMESTAMP
  • TXTIMESTAMP
