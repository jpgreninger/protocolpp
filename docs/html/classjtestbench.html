Detailed Description

Testbench for Protocol++® (ProtocolPP®)

This class inherits from jproducer and implements the platform specific parts of the testbench such as reading and writing memory. It may also include any other methods necessary for the user's platform. The implementation of the jproducer pure virtual functions are found in this class and it's methods and constructor are as follows
