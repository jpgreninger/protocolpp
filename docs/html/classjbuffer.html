jbuffer Class Reference

#include "jbuffer.h"

Detailed Description

Memory using doubly linked-list from the

standard library. Constructs a memory with the

requested number of buffers of the templated size

// create a memory of 256 buffers each 256 bytes long
jbuffer mem1<uint8_t, 256>(256);
// create a memory of 16384 buffers each 384 bytes long
jbuffer mem2<uint8_t, 384>(16384);
// request 20 buffers from mem1
uintptr_t addr1 = mem1.dequeue(20);
// return the buffers to the memory
mem1.enqueue(addr1);
// request 200 buffers from mem2
uintptr_t addr2 = mem2.dequeue(200);
// return the 200 buffers to mem2 from the shared_ptr
mem2.enqueue(addr2);
// return number of available buffers in mem1
int numbufs = mem1.avail();
jbuffer

Using the buffers is simply using the functions for std::array. Iterate through the buffers in the list, write to or reading the data from the memory at each node. For example, to used the addr1 above to write data it would be

mem1.write(addr1, input1, inlen);
mem1.read(addr1, out1, outlen);
